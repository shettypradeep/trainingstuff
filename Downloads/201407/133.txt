You either have to execute it manually with 

mvn checkstyle:check

or if you want it to run as part of a normal build you have to bind it to a lifecycle phase in the config

 <executions>
                        <execution>
                            <phase>process-sources</phase>
                            <goals>
                                <goal>check</goal>
                            </goals>
                        </execution>
                    </executions>

Also ... check out the setup on for reusing the config file without having relative paths.. 

https://github.com/mosabua/ksoap2-android/blob/master/pom.xml
