The Apache Maven team is pleased to announce the release of the Apache =
Maven SCM Plugin, version 1.9

The SCM Plugin offers vendor independent access to common scm commands =
by offering a set of command mappings for the configured scm.

http://maven.apache.org/scm/maven-scm-plugin/
You should specify the version in your project's plugin configuration:

<plugin>
  <groupId>org.apache.maven.plugins</groupId>
  <artifactId>maven-scm-plugin</artifactId>
  <version>1.9</version>
</plugin>
