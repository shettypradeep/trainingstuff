Hi,

Your are probably missing

        <configuration>
          <source>1.7</source>
          <target>1.7</target>
        </configuration>


Basically you are using the 1.7 compiler to compile java5 source code
to java 5 byte code (1.5 is the default value for source/target IIRC)
