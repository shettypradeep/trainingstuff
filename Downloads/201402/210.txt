I am pleased to announce the 1.1.0 release of Reflow Maven skin!

Reflow Maven skin builds on Bootstrap and offers responsive design, modern
web components, JavaScript goodies and wide customisation options. See it
in action and read the documentation at Reflow skin website:

http://andriusvelykis.github.io/reflow-maven-skin/

---

The following is the overview of changes in the 1.1.0 release.

# Major enhancements

- Support for custom/local resources (vs. hardcoded CDN-based jQuery and
Bootstrap), which enables use of Reflow skin in a local/intranet setting

# Minor enhancements

- Upgraded to Bootstrap 2.3
- Bumped versions of other skin components

# Bug fixes

- Fixed ToC snapping when banner is not available/is of custom size
- NPE when menu item href is not set
- Other various fixes (see GitHub issues)

See all GitHub issues for 1.1.0 for further details:

http://github.com/andriusvelykis/reflow-maven-skin/issues?milestone=1&state=closed

While this update does not yet support Bootstrap 3, such upgrade is in
plans and should be the next target of Reflow Maven skin development.

Thanks to everyone who has contributed with issues, suggestions, code and
support.

Enjoy,
~Andrius
