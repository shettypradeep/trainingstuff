scheme of things if the connections are pooled and you're using =
something like SPDY/HTTP2 and a good client. Trying to manage TLS and =
non-TLS connections to the same server would not be worth the complexity =
for likely no performance improvement. Note that all Google servers are =
SPDY-enabled and all modern Android devices will connect using SPDY and =
this all scales and performs well.
